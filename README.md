闲极无聊，基于公开专利顺手推演完成了一套EUV步进式模块化工程方案，整理全套公开推演参数与工程逻辑对外开源。本人只是普通技术爱好者nobody，非从业机构、无任何行业背书，纯属公开技术推演分享。无需抬杠、无需过度解读、无需刻意炒作，业内同仁有兴趣可自行参考验证，无关舆论无需纠缠。
首先要了解到该系统系静态曝光系统即步进重复式光刻机（stepper）模块化改造部分参数 

该系统基于已公开专利 

CN122331213A 一种用于极紫外光源的镜面碎屑防护系统

CN122239382A 一种模块化极紫外光刻机及其改造方法

前提：

该机型属于工程验证机型

其次该系统光源和机台是模块化分离，入射光靠真空波纹管外接抗振动快门控制。

工程设计主要参数如下：

全向euv收集率：15% 
固体激光euv转换率：CE=3%
激光和锡滴喷嘴工作频率：3-10khz （工作频率5khz）
照明反射镜总数：4面 （其中包含左右分光中继）
机台投影物镜镜组：4面 （共轴）
放大倍率：4
镜面反射率不低于65%  
总计10面镜 （但是单一光路对应只有9面反射镜）
单束晶圆入射euv脉冲能量要求不小于0.02mj

NA=0.35

主激光参数：350mj 8MW 30-50ns 3-10khz  (实际使用功率控制在250mj）

主要生产芯片规格：8*8 mm
系统生产率：10-15WPH
系统生产尺寸：小于等于8寸
系统套刻精度：亚纳米

工程难点：反射镜掩膜吸收层厚度最好控制在30nm及以下，需要新材料！！！

建议使用光刻胶：20mj金属胶
预计使用气体：氢气（气流场速度大概在10m/s以下？）（废料收集碗在抽气口的外缘？）

实际操作中可能要进行补曝光（OPC实测验证再进行数据判断），光源拼接计算难度较高（但低于扫描式）

晶圆载台采用SiC载台（需表面硬化）加装压电陶瓷对准装置，以上机械和对准装置均需满足真空生产需求

对准采用 粗对准--检测---精对准
双工件台建议 先进行map构建，然后建立OPC的矫正，然后再进行工作台和扫描台的切换
需配置2台PM级国产干涉仪（精度对准轮替使用，外部需单独加装避震平台和恒温仓，建议在实际产线上操作更换真空中转仓的时候自动切换 0.5-1小时一次！，切换在真空中转仓的切换时间内）
MAP多轴差分干涉仪1台


抓取机构，两台真空机械臂，一台负责抓取晶圆 一台负责更换掩膜

预计费用：
光源部分约1亿-1.5亿人民币 
机台改造新增设备费用约8000万-1亿人民币
联调费用和开发费用约1-1.2亿人民币
预留费用约0.5-1亿人民币
BOM约3.5-4.5亿人民币



整体判断，该机型所有部件均可国产化，因为该机型对于中性碎屑有较好的防治措施，所以该机型体型较小，整体功率不大，热管理措施较好处理，收集镜和部分场景均只需要水冷，生产安装条件相对简单，主要是避震措施需与标准的EUV产线一致，因光源的真空腔体和机台真空腔体均属于独立腔体，对于环境洁净要求可大幅降低，环境工程预算可大幅下降，预计整机功耗约20KWH，外部电力工程预算可大幅下降，整机造价相对较低，整机占地约80平方米，层高约3.5-4米

缺点是生产率较慢，晶圆尺寸小，视场小，生产大尺寸die需要多次拼接，考验套刻精度，指标预计可做到台积电N8节点 FINFET工艺，N5节点需要做多重曝光，晶圆厂必须要配置电子束光刻机制作掩膜

建议主要生产的目标区间 N8节点的中型CHIP，N5节点的CHIPLET和3D堆叠，拼接建议先试生产一次拼接，即8*16 可以保证良率和生产效率（是否只做关键层看具体工艺要求）
